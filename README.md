This repository contains a tutorial for MEG data analysis using the UMD-KIT MEG system. 

To download the example data, please visit this [box link](https://umd.box.com/s/8f4rw6fpgq9muxtcburi34h5zffhgolp) and save the files to the 'R3289' folder within this project. 

The entire tutorial is contained in the PreProcessing.ipynb notebook.
